# Go Workshop Conference 2025 Website

## 開発環境のセットアップ

1. 必要なツールをインストールする
    - 以下のツールをインストールしてください。
      - [Node.js](https://nodejs.org/en/download/package-manager): v22.x
      - [pnpm](https://pnpm.io/installation): v9.x
2. 依存パッケージをインストールする
    - `pnpm install`を実行してください。

## コマンド

- `pnpm dev`: 開発サーバー（<http://localhost:4321/>）を起動する
